最后一个在校上课的学期要结束了

2012-06-02
  • 生活
  • 随笔

每个学期临近期末的时候都会特别繁忙。不用考试的课程，也不像其它专业那样交个论文就行，一般都要求做个大型实验。这个学期学的是 JavaEE ，最后的大作业是用 SSH 框架做个网站。若是以前，我会利用课余时间，更加深入学习本学期的课程。但这学期实在是目标明确的几乎所有专心学习的时间都花在了 iOS 的学习上，所以对这门课的关心投入太少，对这个大作业也是烦的透顶。

最近越来越深切地感受到，每个人都不可避免地要做自己不喜欢做的事。最悲哀的是明明知道自己不喜欢，在做的过程中少不了痛苦，却依然因为要完成任务而不得不硬着头皮做下去。但幸好，也是在这样的对比中，才更加明白自己喜欢做什么。好像忽然分出身来站在自己身边，看到自己在做喜欢的事情的时候的那种原本不自知的专注和满足。

回首即将过去的这个学期，还算是幸福的。除了最后一个星期，开始为大作业烦恼，其他时间都算是在做自己喜欢的事。不过，烦恼也就是在不知道做什么的时候烦恼一下而已。后来花了点时间思考，最后决定为之前已完成的 APP 写服务器端接口然后开工之后，实现的速度还是挺快的。今天上午也终于完成了这个项目的文档，总算结束了这件事。而且整个大学生生涯的课程也在昨天六一儿童节的晚上上完了。算了一下从小学开始读了十四年的书终于在这么一个本该装装嫩却一直在赶文档而且晚上还有课的一天结束了，心中还是很有感慨的。现在剩下的就是学校规定的四个月的实习和毕业设计的事儿，我的学生生涯就要结束了。

最近让大三学生精神紧张的找实习的事儿也算是告一段落了。虽然有些同学还在纠结到底是去那个公司好呢，是看当前的待遇呢还是看在公司长远的发展呢，但终归是找到了份实习。只有我这种看似目标明确，自信满满其实内心也有些许慌张的人还没有一张Offer在手。不过等下周二考完测试就开始投简历吧。刚刚本来只是想完成就业指导课的简历的作业的，没想到最后成了针对我一直很想去的那家公司的专门的简历。到时候得再修改一下就可以投出去了。

看他们找实习的经历，自己也有一些思考。有几个专业牛人之前一直在钻研Java的东西也从不怀疑自己将来会做 Java 开发，但收到某大公司的 Offer 之后，决定把已搁置很久的 C++ 捡起来，还要再学习 C# 。室友本已确定要去的公司笃定要做DBA连三方协议都差不多要签了最后被那个大公司录取，纠结了小段时间也决定投入大公司做 C++/C# 开发。还有人因为没有告知HR直接不去面试，最后那个公司反应到学院要求书面道歉的，他在那纠结到底是不是公司小题大作了，自己有没有必要写道歉信。也有人到现在也还在纠结薪酬问题决定不下而有些人却令人意外地坚定地要去一个工资低于一般水平很多却顶着“规模大”的光环的公司。我在想像我这样脑子里只放了一家公司，而且一定要找 iOS 实习的人到底是该说专注呢还是一根筋。好吧，暂且还是往好的地方想吧，只是在我的臆想之中，那样的我会过得比较快乐。

到了大三下，考试什么的大家真的也没有那么上心了。基本上都算是找到实习了，还有多少人那么在意绩点呢。毕竟对于大部分人来说当初想考个高分是为了现在找工作好找些。但事实证明简历上就算全是高分也不会收到预期的效果。

但是，那个奖学金还是不错的。二等的话还能交两次 iOS Developer Program 的年费。嗯，所以决定明天去图书馆把书啃一下。
