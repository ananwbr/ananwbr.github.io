老妈来电说...

2012-03-04
  • 生活
  • 随笔

到了一周一记的时间。

傍晚接到老妈的电话。很明显，越是接近毕业，讨论的话题就越来越多得设计到将来得职业选择上。在她得观念中，学习成绩好得学生理应被公司直接“请”过去。但现在早已不是大学生值钱到被争夺的年代。虽然也会有优秀到这种程度得学生，但显然，我并不属于此列。每次听她念叨着“要是被（公司）请走就好了”，我都会有一种我要更加努力的想法。

今天和老妈的通话还收获了一点很重要的信息，从小一起长大的但后来上了不同高中再是不同大学而渐渐疏于联络的伙伴居然真的就订婚了，而且已经跟男方一起生活。寒假的时候有些耳闻，但没好意思向她本人证实，也抱着“如果这是真的她应该会通知我们吧”的想法。这好像是第一个定下终身大事的朋友，让我再一次真切的感受到了我们已经长到了可以谈婚论嫁的年纪。可是明明，经历了那么多艰苦奋斗，我才刚要开始目标明确的追求我想要的生活，属于自己的，自由自在的生活。和另一个人生活在一起，意味着要承担更多的责任，是不是太早将自己束缚在“家庭”这个地方了呢。可是，这或许就是她追求的人生吧。

这件事情的另一个点是老妈末了加了一句“那个男生家里很有钱”。我努力揣测她说话的语气。我不知道这句话是不是说明了朋友接下去的幸福生活。我不知道老妈是不是觉得这就是一个女孩子最好的归宿。逢年过节亲戚们聚在一起总是会说到谁谁家的女儿嫁了个有钱人，然后家里的经济条件也跟着改善。我问过老妈是不是心里也总希望我能嫁个有钱人，然后帮着改善家里。虽然她每次都会说只要我幸福就好，虽然就算我没有嫁到大富大贵之家也会尽我所能照顾他们，但是我想作为一个母亲总还是自己的女儿将来能够衣食无忧吧。

老妈和她的朋友现在总会聊到儿女的终身大事，似乎暗中也在比较谁家的女儿能找到更好的归宿。还小的时候，比较的是谁的学习成绩比较好。在这一方面我的确收获了很多夸赞，也让老爸老妈长了脸。可是现在呢，似乎嫁的好才是值得炫耀的事情。

读了十几年的书，希望的就是将来能过上好生活。为了实现这样的愿望，我还在努力奋斗中。我并不是排斥那些因为嫁了个有钱人而一下子变的有优越感的人，因为她们能做到说明也有自己的优势。在这件事情上我的确也有过“将来会令爸妈失望”的顾虑。但是转念一想，还算年轻的我又何必为了这种事情浪费时间呢。我其实是相信缘分的。遇到什么样的人，冥冥之中上天自有安排。而且谁说通过我自己的努力，我不能给自己想要的生活，不能让爸妈过上好生活呢。

大概是因为寒假里听了太多这类事情，再加上朋友的事，有点受刺激了吧。我很清楚有种生活并不适合我，我还会沿着自己设定的方向一如既往的走下去。

还有件事值得说一说。和一高中同学聊天。说到毕业后去留的问题。我让她也留在杭州跟我分担房租吧。她突然说和我一起住会有压力，她说她可以预见我们的工资的差距。虽然目前连我自己都不知道将来会拿怎样的工资，但是我是却感受到从现在开始这么好的朋友之间也会考虑收入差距的问题。说实话，有点小伤感。

但将来不管怎样，我都会努力不会让这种原因影响朋友之间的感情。我不够的我会继续努力。他们有不足的也会提供帮助。

这是我的初衷，虽然不能保证，但至少会一直提醒自己。

本来要总结这周的学习状况的。抒发一下情感也好。

晚安。

又会是一个新的开始

2012-01-28
  • 生活
  • 总结

本打算元旦的时候就要写一篇总结的，后来找借口要自己等到农历年过去再写，而这几天穿梭于各种宴席，纵情于各种聚会，就是静不下心来写点东西。但这种状态显然不是我想要的，总得强制自己可以沉下来想一想。

过去的这个学期里我不知感叹了多少次时间飞逝，但想起开学初做的事又觉得很遥远。和所有的假期一样，上一个暑假我给自己制定了新学期计划。包括学习，锻炼，旅游各个方面。而此时我却不得不承认，这又是一个没有完成计划的学期。

开学初我一门心思扑在 Android 上，一到周末就背个电脑到图书馆找本书边看边敲代码，一坐就是一整天。本来是兴趣满满的，却在看了几本 iOS 的书之后兴奋异常，觉得这才是我想走的方向。那段时间正好碰上软考，可是我却一直忍不住阅读 iOS 的书。好险最后 140 没有白交，证是拿到了。不过我的 Android 学习就此断了。然后开始烦恼是不是要买一台 mac ，想着如果到手了我是否会真心热爱，并投入百分之百的热情，如果我学有所成了移动互联网的热潮是不是会退却了。但显然这些不是我当前该关心的事情。对整个行业来说，这是一个机会。不管成功的几率有多大，不去试一试又怎么会抓住机会呢。我想这样的道理，行业的前辈应该早就明白了。于是我也不再摇摆不定，决定开始自学。

在从 Android 到 iOS 的过渡期间我也曾对开源充满了兴趣。因为上了《操作系统原理》的课程，开始学习 Linux 的知识。最开始读的是《 Linux 内核设计艺术》，这本书让我经常有一种恍然大悟的感觉。比如以前我一直很好奇为什么开机就会运行操作系统，为什么它就能管理其它应用程序。有些知识学通了之后非常简单，可是在那之前却充满了神秘，也令人感到疑惑。以这本书为契机，我又陆续读了其它相关的书籍。也与前个学期的《计算机组成原理》和《计算机网络原理》等课程的知识联系了起来。我第一次有这么深切的感受到原来知识是联通的。以前我会说为什么要学这个，有什么用呢。现在看来要懂得将各种知识联系起来，这样才能看到学习的价值。

现在想想学期的前半部分真是爱折腾。光是 Ubuntu 就不知道装了几遍。本打算放假了要脱离虚拟机将 Windows 换成 Ubuntu 的，但在决定朝 iOS 开发方向努力后，也终于买了 mac ，最后也就将注意力放在 mac OS X了。在折腾 Ubuntu 那会儿，也盲目的学习 vim ，甚至还苦恼我到底是加入 vim 呢还是 emcas 呢。看来我真是浪费了很多时间啊。还想学习建 Linux 的服务器。还爱上了开源，觉得能为开源社区贡献自己的力量真是很有意义的事。当然自己的能力并没有到那种程度，最多也就是个爱学习的人。另外我也有一小段时间在各种开放平台闲逛，在人人的开放平台试了一下水，不过那时候还在 Android 阵营。

这一切直到我决定学习 iOS 就结束了。虽然对很多领域的热情没有持续燃烧下去，但那段时间也都是我作为学生的十几年来觉得最有乐趣的一段时间。不是按照学校规定的课程来学，而是完全按照自己的兴趣去学习，而且在学习的过程中将各种已经学过的孤立的知识串连起来。每天从图书馆回来都觉得这又是充实的一天，因为又有了很多新的收获与感悟。这样的日子，虽然每天依然是自己一个人，但却因为学习感觉幸福。真是神奇。

之前因为陈皓老师要为女程序员写一遍博客，读了很多女性前辈的职场经历与感悟。其实原本从来不觉得女性在职场上会有弱势，我一直相信只要肯努力，有能力，一定会实现自己的价值。但看了各位前辈的文章之后，还是看到了很多辛酸。不过，这对我的职业选择不会有影响，从我选择这个专业开始，我就做好了准备。

我所写的也算是对自己精神喊话吧，希望自己能更加坚定的在这个方向走下去。新的学期也会制定新的计划。也要开始准备实习。这会是充满挑战却令人期待的一个学期。我会努力做到更好。

怎样的我

2011-12-16
  • 生活
  • 随笔

总觉得写一篇文章，需要带点诚心，才有底气拿出来被别人阅读。

很久很久以前，在我还怀着作家梦的时候，刚接触网络，记得在51申请了个账号，心里充满热情地想要在网上留下很多文字，能让更多人读到。但显然，那时候的我只是仗着在学校的作文比赛中得奖累积的信心，也高估了自己的热情，最后不了了之。

自从大学读了工科以后，写文章的功力更是日益衰弱。本来多多少少还是有点做文艺青年的志向，最后在写了越来越多的代码之后，对这事也就放下了。现在想想，写代码也算是一种创作吧，只是这种语言只有计算机能懂，我总是希望他们也能明白其中“遣词造句”的用心。

现在重新拾起博客这事，原因有三。第一，发现自己记性没有想象中好，前段时间翻阅了以前的日记，才感叹“噢，原来还有这种事”，我想，记录生命中更多有意义的时刻，累积起来会是很大的一笔财富吧。第二，在专业学习方面我也是有很强的企图心的，我学习了什么，收获了什么，希望能记录下来，为将来的职业道路也能打点基础。看到很多大牛坚持写了那么多年的博客，不管是关于生活还是技术，都能看出他们对生命的热情。这一点，是非常值得学习的。第三，LOFTER的简约风格，是我非常钟意的。当然还有一个重要的契机，就是上次看到许哲学长在这里的一篇关于创业的文章，想要分享，于是也准备在这筑窝。（补：现在有了自己的博客，就搬过来了:D）

所以，简而言之，以后要在这里留下的，会是生活上的感悟，专业上的收获，当然应该也免不了抽风的时候的碎碎念，比如对某某人的想念。

我是怎样一个人呢？对每个人来说这都是很有难度的问题吧。

我的身体里住着很多个灵魂。

我希望将来过一种很安稳的日子，和心爱的人，开一个小小的咖啡店，店里有手工现磨的咖啡，有满柜子的书，有吃不完的甜点，有贴满纸条的墙。我就坐在店里，等着每一个想要享受慢时光的客人，与他们交谈，对他们微笑。想要离开一阵的时候，可以没有顾虑的背上行囊，上路。不管路上收获的是温暖，还是冷漠，在回到咖啡店后，都能静下心来慢慢沉淀，依然保持对生活的热爱。

我又想趁着年轻，和一群志同道合的人，为这个世界，创造哪怕只是一点点的不同。我希望自己会一直处在这样充满热情的状态，因为我不甘心我来世界上走一遭，却没有留下任何热血的故事。我也总是愿意相信，虽然我们不知道将来会得到什么，但我们知道一定会有惊喜，甚至比我们期望的更多，只要我们能一直努力。

于是，我未来的梦想就这样分了两个时间段。奋斗在前，享受在后。

在别人眼里的我，也是一个矛盾体。有人说“你总是那么自信，让我很有安全感”。有人说“你胆子小，不然会做的更好”。有人说“就你这性格，将来不会有什么出息”。还有人说“你很棒，你是我的骄傲”……好吧，我必需得承认，最后两个通常都是出自我妈的口，前者是在我们吵架的时候，后者是在我取得进步的时候。但我知道，他们都是爱我的。

甚至很多人都觉得我不太像个女生。他们觉得我太独立，不会撒娇，不爱打扮，不爱逛街。我必须得说，有些看法是误读，也说明这一部分人并不是我最贴心的朋友。因为懂我的人都知道，在某些方面，我的确有些执着。我想，那也可以算是一种优点吧。

剖析自己剖析的也差不多了，第一篇的总体目的达到。实际上本来还想要写更多内容，看来可以放到下一篇再啰嗦了。码了这么多字，还是有点小窃喜，发现自己写文章的水平还没有衰退到想象中的那么乱。

虽然我是抱着有读者的心情来写的，但是我从来也不会刻意去经营。毕竟最终的目的还是留给多年后的自己的美好回忆。希望那时的我也足以令自己感到骄傲。

我的人生，未完待续……
